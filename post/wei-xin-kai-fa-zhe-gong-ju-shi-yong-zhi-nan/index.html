2022-11-29

微信开发者工具

一、初步认识开发工具

.wxml：类似于html语言，是小程序一个页面的框架，进行组件的布局
.wxss：用于对小程序的组件进行渲染，如设置颜色、位置等等
.js：是小程序组件功能实现的依托，用于定于函数并进行调用

接下来，我们以一个名片小程序的开发过程为例，对微信开发者工具进行初步认识。

1.1 插入图片

我们首先设计一张名片：

在.wxml文件中插入图片：

<!--index.wxml-->
<image mode="widthFix" src="../../img/Card.png"></image> <!--图片组件-->

在.wxss中进行调整：

/**index.wxss**/
image{
  width:100%;/**使得图片宽度占到页面的100%**/
}

  font-size: 24px;
  font-weight: bold;/*文字加粗*/
  padding-top: 10px;

1.5.1 后代选择器

相当于类中的成员，比如我们在.wxml文件中定义了如下的card类，且类中有一个组件image

<!--pages/card/card.wxml-->
<view class="card">
  <image src="../../img/bg.png" mode="widthFix"></image>
</view>

那么我们可以在.wxss中定义.card image对card类中的组件image进行属性设置，而不会对card外的image起作用：

/* pages/card/card.wxss */
.card{/*card类*/

}
/*后代选择器*/
.card image{/*card类中的image组件*/

}

1.5.2 绝对定位

为了调整logo的位置，我们可以进行绝对定位，在.wxss文件中将logo图片设置为绝对定位，但是还需要将其父元素card设置为相对定位

<view class="card">
  <image class="bg" src="../../img/bg.png" mode="widthFix"></image>
  <image class="logo" src="../../img/logo.png" mode="widthFix"></image>
  <view class="info">
    <view class="name">NExT学院</view>
    <view class="desc">专注高质量IT课程</view>
    <view>1335287851@qq.com</view>
  </view>
</view>

/* pages/card/card.wxss */
.card{/*card类*/
  margin: 10px;/*外边距*/
  position: relative;/*相对定位*/
}
/*后代选择器*/
.bg{/*card类中的image组件*/
  width: 100%;
  border-radius: 5px;/*边框圆角设置*/
  box-shadow: 0 0 5px #5F687E;/*阴影效果*/
  /*x轴偏移量,y轴偏移量,阴影宽度,阴影颜色*/
}
.logo{/*logo类*/
  width: 60px;
  position: absolute;/*绝对定位*/
  top: 10px;/*绝对位置*/
  right: 20px;
}
.info{/*名片信息类*/
  position: absolute;
  bottom: 30px;
  left: 30px;
  font-size: 14px;
  color: #5F687E;
  font-family: "微软雅黑";/*字体*/

1.6 WXSS 入门

**（1）类选择器 class：**类名可以用空格隔开，并且在wxss中后定义的选择器会覆盖掉定义在前面的类选择器的效果，比如我们在名片信息info类中再添加一个类名“red”，并且在wxss中将red类的颜色改为红色，那么此时名片效果就会变为：

<view class="card">
  <image class="bg" src="../../img/bg.png" mode="widthFix"></image>
  <image class="logo" src="../../img/logo.png" mode="widthFix"></image>
  <view class="info red"><!--此处定义了两个类-->
    <view class="name">NExT学院</view>
    <view class="desc">专注高质量IT课程</view>
    <view>1335287851@qq.com</view>
  </view>
</view>

.red{/*一定要带"."*/
  color:red;
}

**（2）组件选择器：**用在wxml中的组件也可以用定义成选择器，比如我们可以将view定义成选择器，设置边框则得到的效果如下：

view{/*此处不需要"."*/
  border: 1px solid #000000;
}

该组件并不常用，因为命中的对象过多。
通式选择器可以进行组合定义，比如我们定义：

image,view{/*此处不需要"."*/
  border: 1px solid #000000;
}

1.8.3 数据绑定

为了避免在wxml中重复得使用各个模块，我们可以使用数据绑定解决该问题：
我们首先删去重复的模块，只保留一个模块：

首先定义data对象：

// pages/card/card.js
const App=getApp()//获取应用实例
Page({//最外层是Page函数，第一层是一个未命名的对象
  data:{//data是上层对象的一个成员，值为一个对象
    price:0,
    name:"机器学习导论",
    coverPath:"../../img/2.png"
  }
})

接着在wxml中修改课程模块处的信息：

<view class="course-card relative">
	<view><image src="{{coverPath}}" mode="widthFix"></image></view><!--图片-->
    <view>{{name}}</view><!--名称-->
    <!--模板条件语句-->
    <view wx:if="{{price>0}}" class="price">{{price}}</view>
    <view wx:else class="price">免费</view><!--免费--><!--注意语法有两个{}-->
</view>

因为我们是有多个课程的，所以将data中的成员改为一个列表，列表的元素是一个对象：

// pages/card/card.js
const App=getApp()//获取应用实例
Page({//最外层是Page函数，第一层是一个未命名的对象
  data:{//data是上层对象的一个成员，值为一个对象
    list:[
      {
        price:0,
        name:"机器学习导论",
        coverPath:"../../img/2.png"
      }
    ]
  }
})

接着我们修改wxml中的语句，利用循环插入多个课程：

<view class="list"><!--遍历list中的元素，将list[i]赋值给item-->
    <view wx:for="{{list}}"><!--注意循环要写在flex模型类里面，否则会导致flex失效-->
        <view class="course-card relative">
        <view><image src="{{item.coverPath}}" mode="widthFix"></image></view><!--图片-->
        <view>{{item.name}}</view><!--名称-->
        <view wx:if="{{item.price>0}}" class="price">{{item.price}}</view>
        <view wx:else class="price">免费</view><!--免费-->
    </view>
</view>

由此我们可以实现之前的效果，而不需要多次重复定义，仅需要在js文件中的list中更新数据即可。

1.8.5 函数

我们在首页界面头部添加两个button，我们希望这两个按钮实现过滤的功能

<view class="head">
	<button size="mini">付费</button>
    <button size="mini">免费</button>
</view>

我们在js文件中定义一个响应点击的函数，并在wxml中利用bindtap进行绑定：

onCost:function(){
    let list=this.data.list;
    let newlist=[];
    for(let i=0;i<list.length;i++){
      if(list[i].price>0) newlist.push(list[i]);
    }
    this.setData({list:newlist});
  },
  onFree:function(){
    let list=this.data.list;
    let newlist=[];
    for(let i=0;i<list.length;i++){
      if(list[i].price==0) newlist.push(list[i]);
    }
    this.setData({list:newlist});//设置数据
  }

<view class="head">
    <button size="mini" bindtap="onCost">付费</button>
    <button size="mini" bindtap="onFree">免费</button>
</view>

我们可以将上述两个函数整合为一个函数onFilter，省去重复的代码：

onCost:function(){
    this.onFilter('cost');
  },
  onFree:function(){
    this.onFilter('free');
  },
  onFilter:function(type){
    let list=this.data.course;
    let newlist=[];
    for(let i=0;i<list.length;i++){
      if(type=='cost'&&list[i].price>0) newlist.push(list[i]);
      if(type=='free'&&list[i].price==0) newlist.push(list[i]);
    }
    this.setData({list:newlist});//设置数据
  }
